# Pointfreeco Tap

## How do I install these formulae?

`brew install pointfreeco/tap/<formula>`

Or `brew tap pointfreeco/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "pointfreeco/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
